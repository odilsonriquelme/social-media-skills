DESCRIÇÃO
---------
Conjunto completo de Claude Skills por trás do sistema de conteúdo do Charlie Hills
(350k+ seguidores em LinkedIn, Instagram, Substack, X e YouTube; 100M+ views/ano).
Skills são arquivos markdown que dão ao agente conhecimento e workflows especializados:
quando você trabalha numa tarefa de redes sociais, o Claude reconhece e aplica os padrões,
regras de voz e restrições de cada plataforma. Instaláveis via marketplace de plugins.

ARQUITETURA IMPORTANTE (skills em CASCATA)
-------------------------------------------
As skills NÃO são independentes — leem um contexto compartilhado, em hierarquia:
   1. voice-builder  → FUNDAÇÃO. Gera about-me.md + voice.md (lidos por TODAS as outras)
   2. newsletter-voice → a FONTE de onde todo conteúdo nasce (newsletter primeiro)
   3. as demais skills → derivam dessa base para cada formato/plataforma
Ou seja: o sistema começa na newsletter e flui para todos os outros canais.

AS 17 SKILLS (skills/)
----------------------
FUNDAÇÃO / VOZ:
   voice-builder      → define a voz/identidade verbal (base de tudo)
   newsletter-voice   → escreve a newsletter (fonte de todo o resto)
   niche-research     → pesquisa de nicho e tendências

ESCRITA / POSTS:
   post-writer        → escreve posts
   post-formatter     → formata posts para a plataforma
   post-scorer        → avalia/dá nota ao post antes de publicar
   hook-generator     → gera hooks (ganchos de abertura)
   quote-post         → posts de citação
   pinned-comment     → comentário fixado (LinkedIn) + prompts de imagem
   content-matrix     → matriz de conteúdo (planejamento multi-formato)

VÍDEO / VISUAL:
   reels-scripting    → roteiros de Reels / vídeos curtos
   youtube-thumbnail  → thumbnails de YouTube
   graphic-designer   → design gráfico de peças
   gemini-carousel    → carrosséis usando Gemini
   gemini-infographic → infográficos usando Gemini

PERFIL / DADOS:
   profile-optimizer  → otimização de perfil/bio nas redes
   analytics-dashboard→ painel de métricas/analytics
